# Yishu Zhu — Academic Homepage

Personal website with white theme and custom color palette.

## Deploy to GitHub Pages

1. Copy the contents of this folder into your [yishuzhu.github.io](https://github.com/yishuzhu/yishuzhu.github.io) repository.
2. **Keep or re-add these assets** from your current repo if you use them:
   - `prp.jpg` — banner image on the home page
   - `CV2025_Yishu Zhu.pdf` — CV link
3. Push to the `main` branch; GitHub Pages will serve the site.

## Local preview

Open `index.html` in a browser, or run a simple server:

```bash
cd yishuzhu.github.io
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Theme colors

- **#66A5ED** — Primary blue (links, nav)
- **#7758A3** — Purple (accents)
- **#F6C45C** — Gold (highlights)
- **#FFAFEB** — Pink (accents)
- **#DB3E8C** — Magenta (strong accent)
